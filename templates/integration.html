Flask Example

Send security events to Secrust using Python Flask. 


import requests

API_KEY = "{{.website.APIKey}}"

requests.post(
    "http://localhost:8080/api/logs",
    json={
        "api_key": API_KEY,
        "ip": request.remote_addr,
        "method": request.method,
        "path": request.path,
        "status": 200,
        "user_agent": request.user_agent.string
    }
)
